Hide Any Menu User Wise

This module empowers administrators to customize the Odoo interface on a per-user basis by selectively hiding menus for specific users. It enables precise control over what each user sees, without changing their access rights or modifying user group settings.

Key Features

  • Per-User Menu Customization:Easily hide any menu item for an individual user while keeping it visible for others.
  • Group Settings Remain Intact: Works independently of group-level permissions â€” no need to alter default group configurations.
  • Simple Configuration Options: Control visibility directly from a user profile or through a dedicated menu visibility management view.
  • Restrict Access Effectively: Ensure that users only interact with menus relevant to their role, improving both security and clarity.
  • Cleaner Interface for Users: Helps reduce visual clutter by removing unnecessary or unrelated menus for each user.

Create New Hide Specific Menu

Go to Settings > Users and there you can see the 'Hide Specific Menu' tab where you can select any Menu/Submenu that needs to be hidden to the selected user.

Select Menu

Click on 'Add a line' and select the menus that needs to be hidden for this user.

As you can see the menus and corresponding views are now hidden for the user.
