Hide Menu User Wise

Certain menus in Odoo should be restricted to administrators only. These menus can be configured from the Users form view by assigning the appropriate access rights. Only users with Admin-exclusive access will be able to view and use these menus, while non-admin users will not have access.

Highlighted Features

Role-Based Menu Visibility

Menus are displayed only to users with the Admin-exclusive access group.

Non-admin users will not even see the restricted menus in their navigation.

Configurable from User Form

Access rights can be easily assigned from the Users form view without code changes.

Admins can enable/disable access for specific users as needed.

Clean User Interface

Non-admin users see only the menus relevant to their roles.

Simplifies navigation and improves usability.
